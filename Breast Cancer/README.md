# Project Overview

The Breast Cancer Diagnostic System is a Machine Learning-based web application developed using Python and Streamlit. It predicts whether a breast tumor is **Benign** or **Malignant** based on medical features provided by the user.

The application uses the **Wisconsin Breast Cancer Dataset** available in Scikit-learn and employs a **Random Forest Classifier** to make predictions with high accuracy.

---

# Project Scope

This project aims to:

- Predict breast cancer diagnosis using Machine Learning.
- Assist in early detection of breast cancer.
- Provide a simple and interactive web interface.
- Demonstrate the practical application of Machine Learning in healthcare.
